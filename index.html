JavaScript

Es un lenguaje de programación que se utiliza para que las paginas sean mas dinamicas.
La gran diferencia de html y js es que en JavaScript tine constantes, variables y funciones.
Las constantes son elementos que tiene un valor pre determinado en cambio las variables son elementos de un valor que puede cambiar.
Algunos ejemplos number que es una variable de numero, Strings quye sirve como cadena de texto es otra variable,
Las funciones nos permiten agrupar líneas de código en tareas con un nombre, para que, posteriormente, podamos hacer referencia a ese nombre para realizar todo lo que se agrupe en dicha tarea.
Hay dos tipos de funciones declarar y ejecutar.
Ejemplos console.log(): muestra un mensaje o valor en la consola del navegador, alert(): muestra un cuadro de diálogo con un mensaje en el navegador.
DOM es un interfaz de programacion que representa la estructura html y eso nos permite manipularlos dentro del JS.
Ejemplos getElementById, getElementByclassname, getelementbytagname.

API

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permite que diferentes aplicaciones se comuniquen entre sí.
APIs Web: Son APIs que permiten la comunicación entre aplicaciones a través de Internet. Pueden ser públicas o privadas y se utilizan para acceder a servicios en línea.
APIs de Biblioteca: Son APIs que proporcionan conjuntos de funciones y procedimientos que pueden ser utilizados por programas escritos en el mismo lenguaje de programación.
APIs de Sistema Operativo: Permiten que las aplicaciones accedan a recursos del sistema operativo, como archivos y hardware.
- Endpoint: En una API, un endpoint es una URL específica que representa un recurso o una función.
- Autenticación: Para acceder a muchas APIs, es necesario autenticarse mediante tokens, claves de API o credenciales de usuario.
- Solicitud y Respuesta: Para interactuar con una API, una aplicación hace una solicitud HTTP (por ejemplo, GET) al endpoint correspondiente y recibe una respuesta en formato JSON, XML u otro formato.
La función fetch es una función de JavaScript que se utiliza para realizar solicitudes de red, generalmente para obtener recursos desde un servidor web.
1. **Realizar Solicitudes de Red:** **`fetch`** se utiliza para realizar solicitudes HTTP (GET, POST, PUT, DELETE, etc.) a una URL específica. Puedes especificar la URL de destino como argumento de la función **`fetch`**.
2. **Solicitud Asíncrona:** La función **`fetch`** realiza solicitudes de manera asíncrona, lo que significa que no bloquea la ejecución del resto del código mientras espera la respuesta del servidor. Esto es esencial para evitar que la interfaz de usuario se congele mientras se espera una respuesta del servidor.
3. **Promesas:** **`fetch`** devuelve una promesa. Una promesa es un objeto que representa un valor que puede estar disponible ahora, en el futuro o nunca. Puedes utilizar los métodos **`then`** y **`catch`** en la promesa devuelta por **`fetch`** para manejar la respuesta de la solicitud.
